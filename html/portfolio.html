Coding Projects

Minecraft

Scaled Down Version of Minecraft built using OpenGL and GLSL. Incorporates custom features such as AI bird flocks that implement the boiding algorithm and procedural terrain generation with various biomes

Mini Maya

A scaled down version of the Maya editing software. Uses OpenGL and GLSL to create a Mesh with the ability to add vertices and manipulate JSON 3D model files. Also allows for meshes to be attached to skeletons and moved around through linear transformations

Penn Crime Mapper

Using python scripts, we analyzed 60 days of Crime at Penn to see how crimes were distributed throughout the day, geographically located, and what kind of crimes were happening. We then created a frontend site on heroku to map our findings

Anonymous Communications Network

This project was done for the University of Pennsylvania Distributed Systems Laboratory. It is an anonymous communications system designed to hide user metadata from a wide range of adversaries.



PAC Mission-Tracker

This is a web app designed for the Penn Aerospace Club's Balloon team. Our team conducts high altitude balloon launches. This app takes in data from an iridium satellite regarding the balloon's altitude, speed, location, and creates a map representation of that data.
